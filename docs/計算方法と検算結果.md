# てんから勤怠システム — 計算方法と検算結果

## 1. 労働時間の計算

### 計算ルール
- **計算単位**: 1分単位（秒は切り捨て）
- **出勤・退勤**: 打刻時刻をそのまま使用（丸めなし）
- **休憩**: 休憩開始〜休憩終了の実時間を差し引き
- **実労働時間** = 退勤 − 出勤 − 休憩合計

### 残業・深夜・休日の判定
| 区分 | 条件 | 割増率 |
|------|------|--------|
| 通常 | 1日8時間以内 | 1.0× |
| 残業 | 1日8時間超の部分 | 1.25×（基本給1.0 + 割増0.25） |
| 深夜 | 22:00〜翌5:00の実労働 | 1.25×（基本給1.0 + 割増0.25） |
| 休日 | 週の6日目以降（水曜起算） | 1.25×（基本給1.0 + 割増0.25） |

※残業と深夜が重なる場合、二重計算せず深夜を優先（割増は1.25のまま）

### 先方データとの差異（時間丸め）
先方のタイムカードシステムは時間を丸めて集計しているため、当アプリ（1分単位）との間に差が出ます。

**例: 三島恵里加さん 2026年1月**
| 日付 | 出勤 | 退勤 | 当アプリ(1分) | 先方(丸め後) |
|------|------|------|-------------|------------|
| 1/7 | 17:54 | 21:08 | 3:14 | — |
| 1/14 | 17:49 | 20:57 | 3:08 | — |
| 1/21 | 17:44 | 21:34 | 3:50 | — |
| 1/28 | 17:48 | 20:43 | 2:55 | — |
| **合計** | | | **13:07** | **13:00** |
| **基本給** | | | **¥17,052** | **¥16,900** |

→ 差額 ¥152（7分 × 1,300円/60 の差）

## 2. 支給額の計算

### 時給制スタッフ
```
基本給     = 時給 × 全労働時間（時間）
残業手当   = 時給 × 0.25 × 残業時間（時間）
深夜手当   = 時給 × 0.25 × 深夜時間（時間）
休日手当   = 時給 × 0.25 × 休日時間（時間）
交通費     = 日額 × 出勤日数（該当者のみ）
支給総額   = 基本給 + 残業手当 + 深夜手当 + 休日手当 + 交通費
```

### 特別時給（正月・GW等）
特定期間中は全スタッフの時給を一律増額（加算方式）。
```
例: 正月手当 2026/01/01〜2026/01/03 +100円
→ 通常時給1,350円のスタッフは、この3日間は1,450円で計算
```
